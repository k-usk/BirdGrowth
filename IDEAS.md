# BirdGrowth 未来のアイディア帳 🐣

ここは「いつか実現したいこと」「鳥ともっと仲良くなるためのアイディア」を自由に書き溜める場所です。
※ここに書かれた内容は、すぐに実装されるわけではありません。

---

## 🐣 新しい鳥・バリエーション
- [ ] すべての鳥を実在しない鳥に変更したい（鳥好きからもクレームが出ないように）
- [ ] 珍しい色の鳥（パステルカラー、レインボーなど）
- [ ] 季節限定の姿（冬のモフモフ、夏の麦わら帽子）

## 🌿 環境・背景
- [ ] 季節の移ろい（春は桜、秋は紅葉が舞い込む）
- [ ] ネストのカスタマイズ（おもちゃ、止まり木の追加）

## ✨ 演出・エフェクト
- [ ] 成長の瞬間を祝うパーティクル（キラキラ、紙吹雪）

## 🛠 システム・機能
- [ ] GitHub Actionsでセキュリティチェックをしたい
- [ ] 歴代の成鳥たちの「きろく（図鑑）」
- [ ] ウィジェット対応（ホーム画面でいつでも鳥を確認）
- [ ] 歩数計との連動（ヘルスキット連携）
